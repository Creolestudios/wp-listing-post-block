## Description

WP Listing Post Block is the Gutenberg plugin for displaying latest posts with featured image, post title, post content and published date. This plugin consists of one Gutenberg block.

In addition, it has the following features:

- Toggle for show/hide featured image.
- Sorting (newest to oldest, oldest to newest, A->Z and Z->A).
- Number of posts to show.
- Filter out posts by category.

## Installation

1 ) Upload the plugin files to the `/wp-content/plugins/team-member-slider-block` directory, or upload the plugin through the WordPress plugins screen directly.

2 ) Activate the plugin through the \'Plugins\' screen in WordPress
